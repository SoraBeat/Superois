Bienvenidos a SupeRois - Fighting Game Combo Manager
===================================================

¡Gracias por usar SupeRois!
Esta versión "Standalone" no requiere Python y puede ser ejecutada directamente en cualquier PC.

Instrucciones de Uso:
1. Simplemente haz doble clic en `SupeRois.exe` para abrir el programa.
2. IMPORTANTE: Para que la inyección de teclado sea enviada y detectada correctamente por emuladores como Fightcade 2 (FBNeo), haz clic derecho sobre `SupeRois.exe` y selecciona "Ejecutar como Administrador". Esto salta las restricciones de seguridad de Windows.

Carpetas de Datos:
- \config\  -> Contiene tu configuración `settings.json` y la base de datos de juegos `games.json`.
- \data\    -> Aquí se guardan todos los combos divididos por juego (`combos.json`).
** Nota: Si vas a mover SupeRois a otra PC o hacer un backup, asegúrate de llevar estas carpetas porque ahí están tus personajes y combos guardados.

Soporte Gamepad Virtual (Opcional):
Si prefieres que los macros operen silenciosamente incluso cuando no estás usando el programa en 1er plano, o el teclado sigue dando problemas de inyección extraños, puedes usar el Modo Gamepad Virtual:
1. Instala el driver oficial de ViGEmBus en tu PC.
2. Abre SupeRois (indicará "vgamepad: OK" arriba a la derecha).
3. Mapea los controles en el Player 1 de Fightcade al Joystick de Xbox 360 que aparecerá en tu PC.
